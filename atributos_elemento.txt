Prefixo:
	- se o namespace da schema for o default (xs), não é preciso prefixar todos os elementos da XML Schema


<schema>:

attributeFormDefault / elementFormDefault:
	- default: unqualified ----------------------------------------------------------------------------------------------------
	- se o valor for:
		- unqualified: atributos/elementos do target namespace não precisam de prefixo (podem ou não ter)
		- qualified: atributos/elementos do target namespace precisam de prefixo
	- atributos/elementos individuais podem dar override a isto para o seu scope local com o atributo form

blockDefault:
	- The block attribute prevents a complex type (or element) that has the specified type of derivation from being used in place of the inherited complex type (or element)

finalDefault:
	- define o valor default do atributo final em todos os elementos element, simpleType e complexType do target namespace da schema
	- valores aplicam-se a:
		- elementos element e complexType: #all, extension, restriction
		- elementos simpleType: #all, restriction, list, union

id:
	- obrigatório ser do tipo ID e único no documento

targetNamespace:
	- referência URI do namespace da schema
	- pode-se definir um prefixo para o namespace (com outra namespaceDefinition)
	- se não se definir nenhum prefixo, os componentes da schema do namespace podem ser usados sem prefixo

version
xml:lang






<element>:

abstract:
	- se for true, o elemento não pode aparecer na instância do documento, mas sim outro elemento com o substitutionGroup = name deste elemento deve aparecer no seu lugar; pode ser referenciado por mais que um elemento no substitutionGroup
	- default: false ----------------------------------------------------------------------------------------------------

default:
	- mutualmente exclusivo com fixed ----------------------------------------------------------------------------------------------------
	- só permitido em elementos com tipo simples e de texto

substitutionGroup:
	- tem de ter o mesmo tipo ou um derivado do tipo do elemento referenciado
	- só pode referenciar elementos hierarquicamente acima (pode referenciar qualquer elemento da schema)

final:
	- só é permitido em elementos de schema ----------------------------------------------------------------------------------------------------

form:
	- default: valor do atributo elementFormDefault do elemento da schema
	- se o valor for:
		- qualified: este elemento requer um prefixo de namespace
		- unqualified: este elemento não requer um prefixo de namespace (pode ou não ter)

id:
	- obrigatório ser do tipo ID e único no documento

maxOccurs:
	- não pode ser inferior a minOccurs ----------------------------------------------------------------------------------------------------
	- proibido em elementos de schema ----------------------------------------------------------------------------------------------------

minOccurs:
	- proibido em elementos de schema ----------------------------------------------------------------------------------------------------

name:
	- obrigatório se for elemento de schema ----------------------------------------------------------------------------------------------------
	- mutuamente exclusivo com ref ----------------------------------------------------------------------------------------------------

nillable:
	- default: false ----------------------------------------------------------------------------------------------------
	- se for true, permite que a instância do elemento no XML tenha o atributo nil="true"

ref:
	- proibido se for elemento de schema ----------------------------------------------------------------------------------------------------
	- pode ter um prefixo de namespace
	- se existir, não pode ter os elementos complexType, simpleType, key, keyref ou unique
	- se existir, não pode ter os atributos nillable, default, fixed, form, block e type ----------------------------------------------------------------------------------------------------

type:
	- o valor tem de ser um dos tipos built-in ----------------------------------------------------------------------------------------------------
	- ou o nome do atributo type de um elemento simpleType/complexType definido nesta schema ou noutra indicada pelo namespace
	- mutualmente exclusivo com ref ----------------------------------------------------------------------------------------------------
